# Roadmap Visuals Assets

This directory contains visual templates for transformation roadmap presentations.

## Recommended Visual Assets

### Gantt Chart Template
- Timeline visualization showing initiatives across 12-18 months
- Color-coded by phase (Phase 1-4)
- Shows dependencies and parallel initiatives
- Highlights quick wins and milestones

### Radar Chart Template
- Four-axis visualization (one per phase)
- Shows current score vs. peer average
- Highlights gaps and strengths

### Investment Waterfall Chart
- Quarterly investment breakdown
- Cumulative investment line
- Color-coded by investment category

### ROI Timeline Chart
- Dual-axis: investment (bars) and projected value (line)
- Shows break-even point
- Highlights value milestones

### Maturity Progression Chart
- Shows progression from current to target state
- Quarterly maturity score projections
- Comparison against typical progression patterns

### Phase Comparison Bar Chart
- Side-by-side comparison of four phase scores
- Organization vs. peer average vs. leader benchmark
- Clear visual of gaps

## File Formats
Store visual templates in formats that can be easily customized:
- PowerPoint (.pptx) - for editable charts
- PNG (.png) - for high-quality static images
- SVG (.svg) - for scalable vector graphics

## Usage
These visuals should be embedded in:
1. Executive presentation (assessment_template.md section 1)
2. Detailed report (all relevant sections)
3. Standalone roadmap visualizations

## Customization Guidelines
- Use organization's brand colors where possible
- Maintain consistency across all visuals
- Ensure readability at presentation scale
- Include clear legends and labels
- Highlight critical information with visual emphasis
