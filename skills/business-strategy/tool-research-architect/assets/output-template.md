## Notes on Using This Template

**For Simple Research (1-5 tools):**
- Tool Profiles section is sufficient
- Optional: Light summary analysis
- Skip: Complex comparison tables

**For Comprehensive Research (5-15 tools):**
- Full template including summary analysis
- Include: LLM landscape, pricing patterns
- Optional: Feature comparison table

**For Strategic Reports (15+ tools or client-facing):**
- Use full template
- Include: All summary sections
- Add: Recommendations and implementation considerations
- Format: Professional presentation quality

**Formatting Guidelines:**
- Use markdown headers (##, ###) consistently
- Bold all section labels (**Pricing Model:**, **LLM Connection:**)
- Use bullet points for lists, not numbered lists within sections
- Include dates on all pricing and time-sensitive information
- Maintain consistent structure across all tool profiles
- Use tables for comparisons with 4+ tools
- Keep language clear and jargon-free throughout

**Quality Checklist Before Delivery:**
- [ ] All 6 sections complete for each tool
- [ ] Pricing includes verification dates
- [ ] LLM connections categorized accurately
- [ ] Descriptions are jargon-free
- [ ] No placeholder text (TODO, [fill in], etc.)
- [ ] Consistent formatting throughout
- [ ] Sources cited where applicable
- [ ] Executive summary reflects findings (if included)
- [ ] Recommendations are specific and actionable (if included)
