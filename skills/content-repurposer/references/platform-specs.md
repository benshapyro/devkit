# Platform Specifications

Technical specifications for content across major platforms.

## Video Specifications

### YouTube

| Format | Dimensions | Aspect Ratio | Max Length | File Size |
|--------|------------|--------------|------------|-----------|
| Standard Video | 1920x1080 (1080p) | 16:9 | 12 hours | 256 GB |
| 4K Video | 3840x2160 | 16:9 | 12 hours | 256 GB |
| Shorts | 1080x1920 | 9:16 | 60 seconds | 256 GB |
| Thumbnail | 1280x720 | 16:9 | N/A | 2 MB |

**Recommended:**
- Format: MP4 (H.264 codec)
- Audio: AAC, 48kHz stereo
- Frame rate: 24-60 fps

## Audio Specifications

### Podcast Platforms (General)

| Spec | Recommendation | Notes |
|------|----------------|-------|
| Format | MP3 | Universal compatibility |
| Bitrate | 128 kbps (mono), 192+ kbps (stereo) | Balance quality/size |
| Sample rate | 44.1 kHz | Standard |
| Channels | Mono (solo), Stereo (interviews) | Mono = smaller files |
| Loudness | -16 LUFS (stereo), -19 LUFS (mono) | Per Spotify/Apple |
| Max file size | 200 MB typical | Varies by platform |

## File Naming Conventions

### Recommended Structure

```
[platform]_[content-type]_[topic]_[date].[ext]

Examples:
instagram_reel_marketing-tips_2024-01-15.mp4
linkedin_carousel_seo-guide_2024-01-15.pdf
youtube_thumbnail_podcast-ep47_2024-01-15.jpg
twitter_quote_productivity_2024-01-15.png
```

### Version Control

```
[filename]_v[version].[ext]

Examples:
instagram_reel_marketing-tips_2024-01-15_v1.mp4
instagram_reel_marketing-tips_2024-01-15_v2.mp4 (revised)
instagram_reel_marketing-tips_2024-01-15_v2-final.mp4
```

## Export Checklists

### Video Export Checklist

- [ ] Correct dimensions for platform
- [ ] Aspect ratio matches (16:9, 9:16, 1:1)
- [ ] Duration within platform limits
- [ ] File size within limits
- [ ] Captions burned in (for social) or uploaded separately
- [ ] Audio levels normalized (-16 to -14 LUFS)
- [ ] Thumbnail prepared (if applicable)
- [ ] Safe zones checked (for vertical video)

### Image Export Checklist

- [ ] Correct dimensions
- [ ] Resolution 72+ dpi for web
- [ ] File size under platform limit
- [ ] Color mode: sRGB
- [ ] Text readable at thumbnail size
- [ ] Branding included
- [ ] Alt text prepared (for accessibility)

### Audio Export Checklist

- [ ] Format: MP3 or platform-preferred
- [ ] Bitrate: 128-192 kbps
- [ ] Loudness: -16 LUFS (or platform spec)
- [ ] ID3 tags complete (for podcasts)
- [ ] File size reasonable
- [ ] Chapter markers (if applicable)
