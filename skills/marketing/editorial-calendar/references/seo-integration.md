# SEO Integration Guide

Integrate search engine optimization into content planning from the start.

## Keyword-First Planning

### The Process

```
1. Keyword Research
       ↓
2. Cluster Mapping
       ↓
3. Content Matching
       ↓
4. Brief Creation
       ↓
5. Production
       ↓
6. Optimization
       ↓
7. Performance Tracking
```

## Phase 1: Keyword Research

### Research Sources

| Source | Best For | Limitations |
|--------|----------|-------------|
| Google Keyword Planner | Volume data, ad-focused | Grouped ranges, not exact |
| Ahrefs/SEMrush | Comprehensive data, difficulty | Paid, learning curve |
| Google Search Console | Your current rankings | Only existing content |
| Competitor Analysis | Gap identification | Requires tools |
| Customer Interviews | Intent understanding | Qualitative only |
| Social Listening | Trending topics | Not search-specific |

## Phase 2: Search Intent Classification

### Intent Types

| Intent | User Goal | Content Type | Example Query |
|--------|-----------|--------------|---------------|
| **Informational** | Learn something | Blog, guide, tutorial | "what is content marketing" |
| **Navigational** | Find specific page | Homepage, product page | "hubspot login" |
| **Commercial** | Research before buying | Comparison, review | "best email marketing tools" |
| **Transactional** | Make a purchase | Product page, pricing | "buy mailchimp subscription" |

### Intent Signals

**Informational:**
- "how to", "what is", "why does"
- "guide", "tutorial", "examples"
- "tips", "ideas", "ways to"

**Commercial:**
- "best", "top", "vs", "comparison"
- "review", "alternative to"
- "[product] for [use case]"

**Transactional:**
- "buy", "price", "cost", "discount"
- "free trial", "demo", "sign up"
- Brand + product names

### Matching Content to Intent

| Intent | Content Format | Length | CTA Type |
|--------|---------------|--------|----------|
| Informational | Blog, guide, video | Long-form (1500+) | Related content, newsletter |
| Commercial | Comparison, review | Medium (1000-2000) | Free trial, demo |
| Transactional | Landing page, pricing | Short, focused | Purchase, sign up |

## On-Page SEO Checklist

### Title & Meta
- [ ] Primary keyword in title (front-loaded if natural)
- [ ] Title under 60 characters
- [ ] Meta description includes keyword
- [ ] Meta description has clear CTA
- [ ] URL is short and includes keyword

### Content
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Secondary keywords in H2s where natural
- [ ] Keyword density 1-2% (natural usage)
- [ ] LSI terms included naturally
- [ ] No keyword stuffing

### Structure
- [ ] Logical heading hierarchy (H1 → H2 → H3)
- [ ] Short paragraphs (2-4 sentences)
- [ ] Bullet/numbered lists where appropriate
- [ ] Table of contents for long content (1500+ words)

### Media
- [ ] At least one image with alt text including keyword
- [ ] Images compressed for page speed
- [ ] Video embedded if relevant
- [ ] Custom graphics/diagrams where helpful

### Links
- [ ] X internal links to related content (minimum 3)
- [ ] X external links to authoritative sources (minimum 2)
- [ ] Internal links use descriptive anchor text
- [ ] Broken link check before publish

### Content Refresh Triggers

Refresh content when:
- Ranking drops 5+ positions
- Traffic declines 20%+ month-over-month
- Information becomes outdated
- New competitor content outranks you
- 6+ months since last update (for competitive keywords)

## Tools Integration

### Recommended Tools

| Purpose | Free Option | Paid Option |
|---------|-------------|-------------|
| Keyword Research | Google Keyword Planner, Ubersuggest | Ahrefs, SEMrush |
| Rank Tracking | Google Search Console | Ahrefs, SEMrush |
| Content Optimization | Hemingway Editor | Clearscope, SurferSEO |
| Technical SEO | Google PageSpeed Insights | Screaming Frog |
| Competitor Analysis | - | Ahrefs, SEMrush |

### Tool Workflow

```
Keyword Research (Ahrefs/SEMrush)
        ↓
SERP Analysis (Manual + Tools)
        ↓
Content Brief (Template)
        ↓
Writing (Google Docs)
        ↓
Optimization (Clearscope/SurferSEO)
        ↓
Publish (CMS)
        ↓
Track (Search Console + Ahrefs)
```
