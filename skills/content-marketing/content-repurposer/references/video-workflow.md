## Step 2: Clip Extraction

### Clip Selection Criteria

**High-Performing Clips Have:**
- Strong hook in first 3 seconds
- Self-contained (makes sense standalone)
- Clear audio
- Good visual framing
- Actionable or emotional content
- Natural ending point

**Clip Types:**

| Type | Duration | Best For | Platform |
|------|----------|----------|----------|
| Teaser | 15-30 sec | Promotion | Stories, Twitter |
| Insight | 30-60 sec | Value | Reels, Shorts, TikTok |
| Tutorial | 60-90 sec | How-to | Reels, Shorts |
| Story | 60-90 sec | Engagement | All platforms |
| Quote | 15-30 sec | Authority | Stories, LinkedIn |

### Editing for Different Platforms

**YouTube Shorts (Vertical)**
```
Dimensions: 1080x1920 (9:16)
Duration: 15-60 seconds
- Crop/reframe for vertical
- Add captions (burned in)
- Strong hook in 3 sec
- End with CTA or loop point
- Use trending sounds (optional)
```

**Instagram Reels (Vertical)**
```
Dimensions: 1080x1920 (9:16)
Duration: 15-90 seconds
- Same as Shorts
- Add trending audio
- Use native editing features
- Include text overlays
- CTA at end
```

**TikTok (Vertical)**
```
Dimensions: 1080x1920 (9:16)
Duration: 15-60 seconds (up to 10 min)
- Hook-heavy content
- Trending sounds/music
- Native captions or burned in
- Trend participation if relevant
```

**LinkedIn Video (Horizontal or Square)**
```
Dimensions: 1920x1080 (16:9) or 1080x1080 (1:1)
Duration: 30-90 seconds optimal
- Professional context
- Captions essential (silent autoplay)
- Clear business value
- End with CTA
```

**Twitter Video (Horizontal)**
```
Dimensions: 1280x720 or 1920x1080 (16:9)
Duration: 30-60 seconds optimal
- Hook in first 2 sec
- Captions (silent autoplay)
- Punchy, concise content
- Works without audio
```

## Step 3: GIF Creation

### GIF Use Cases

- Reaction moments
- Product demonstrations
- Before/after reveals
- Key quotes (text overlay)
- Tutorial steps
- Celebration moments

### GIF Specifications

| Platform | Max Size | Duration | Dimensions |
|----------|----------|----------|------------|
| Twitter | 15 MB | 5-10 sec | Up to 1280x1080 |
| Slack/Discord | 10 MB | 3-5 sec | Varies |
| Email | 1-2 MB | 2-5 sec | 480-600px wide |
| Blog embed | 5 MB | 3-10 sec | Match content width |

### GIF Creation Workflow

1. **Identify moment** (3-10 seconds max)
2. **Export video clip** at that timestamp
3. **Convert to GIF** (Giphy, EZGif, Photoshop)
4. **Optimize file size** (reduce colors, dimensions)
5. **Add branding** if appropriate
6. **Host** (Giphy for embed, direct upload for social)

### Tools

- **Giphy** - Create, host, share
- **EZGif** - Browser-based conversion
- **ScreenToGif** - Screen recording to GIF
- **Photoshop** - Professional editing
- **Kapwing** - Online editing
